# Emacs Org Mode (CotEditor theme)

File:

- `coteditor/Emacs-Org-Mode.cottheme`

## Import

CotEditor theme files use the `.cottheme` extension and can be imported from the app’s Appearance settings.

Import steps:

1. Open **CotEditor**
2. Open **Settings**
3. Go to **Appearance**
4. Click the **…** (more) button and choose **Import**
5. Select `Emacs-Org-Mode.cottheme`

## Theme format

A `.cottheme` file is a JSON dictionary that contains keys like `background`, `text`, `selection`, `lineHighlight`, and syntax-category keys like `keywords`, `strings`, `comments`, `types`, etc.
